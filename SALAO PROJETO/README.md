# Cisealux Galants - Sitio Web del Salón

Sitio web completo para el salón de belleza unisex "Cisealux Galants" ubicado en Madrid, España.

## Características

- Diseño responsive y moderno
- Sistema de agendamiento de citas online
- Página principal con servicios destacados
- Página "Quiénes Somos" con biografía del fundador y equipo
- Página de contacto con formulario y FAQ
- Paleta de colores personalizada según especificaciones
- Espacios designados para fotos del salón y trabajos
